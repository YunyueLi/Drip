One agent replaces a 5-person UA team.

Open-source autonomous user-acquisition agent for anime / gacha mobile games. Generates creatives, simulates the audience, runs the campaigns, reports the ROAS.

→ Read the code on GitHub

Quickstart

uv pip install drip-agent
cp .env.example .env  # set ANTHROPIC_API_KEY at minimum
drip demo             # dry-run against the bundled spec
drip launch \
  --game ./examples/demo_game.yaml \
  --budget 500 \
  --regions jp,sg,tw

Why Drip

  • Platform AI (Meta GEM, AppLovin AXON, TikTok Smart+) has won the bidding layer — the new battleground is cross-platform orchestration and the creative pipeline.
  • Sett raised $30M Series B as a closed-source SaaS. There is no credible open-source equivalent.
  • Anime / gacha titles burn $5B+ a year on UA but still hand-craft every creative. This is the niche where one agent really does replace five people.

Stack

  • Orchestrator — Anthropic claude-agent-sdk, supervisor pattern over the worker pool.
  • Image — OpenAI gpt-image-2 for keyframes; multilingual text rendering matters for anime CTAs.
  • Video — ByteDance Seedance 2.0 via Volc Engine ARK at ~$0.14/s.
  • Simulation — CAMEL-AI OASIS · up to 1M synthetic agents, Apache-2.0.
  • Ads — Meta + TikTok via MCP (pipeboard-co/meta-ads-mcp, amekala/ads-mcp).
  • Observability — self-hosted Langfuse.
