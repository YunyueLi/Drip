Quickstart

uv pip install drip-agent
cp .env.example .env       # ANTHROPIC_API_KEY at minimum

drip bench list            # 10 UA decision cases
drip bench show 1          # inspect one
drip bench run --agent claude
drip demo                  # end-to-end agent dry-run

Every run is a reproducible bundle. Anyone with the manifest can re-run it and diff the score.

Why Drip

  • Platform AI (Meta GEM, AppLovin AXON, TikTok Smart+) won the bidding layer. The new battleground is the decision layer — what to scale, when to pause, how to diagnose.
  • A wave of closed-source UA agents — Sett, Kohort, GrowthGPT — all claim "70% lower CPA." None publish the cases that produced those numbers.
  • The agent stack is finally good enough. Claude Agent SDK, Meta's Ads CLI, MCP Connectors. The right move now is a transparent reference implementation, not another black box.

Stack

  • Orchestrator — Anthropic claude-agent-sdk, supervisor pattern over the worker pool.
  • Bench — drip.eval: case loader · scorer · LLM-as-judge · reproducible run bundles.
  • Image — OpenAI gpt-image-2 for keyframes, swappable.
  • Video — ByteDance Seedance 2.0 via Volc Engine ARK at ~$0.14/s, swappable.
  • Simulation — CAMEL-AI OASIS · up to 1M synthetic agents, Apache-2.0.
  • Ads — Meta + TikTok via MCP (pipeboard-co/meta-ads-mcp, amekala/ads-mcp).
  • Observability — self-hosted Langfuse.
