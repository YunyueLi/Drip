开源的 UA agent 参考实现。

每一个决定,都把"为什么"摊开给你看。任意 LLM、任意广告平台、任意行业。自带一份开放评测集,任何 agent 都能跑——包括闭源的同行。

→ 在 GitHub 上读源码

快速开始

uv pip install drip-agent
cp .env.example .env       # 至少配 ANTHROPIC_API_KEY

drip bench list            # 10 道 UA 决策题
drip bench show 1          # 看其中一道
drip bench run --agent claude
drip demo                  # 跑一遍 end-to-end agent

每次 run 都是一份可复现的 bundle。任何人拿到 manifest 都能复现并 diff 分数。

为什么是 Drip

  • 平台 AI(Meta GEM、AppLovin AXON、TikTok Smart+)赢下了出价层。新战场是决策层——该扩什么、何时该停、异常怎么诊断。
  • 一拨闭源 UA agent——Sett、Kohort、GrowthGPT——都讲"CPA 降 70%"。没有一家公布那个数字背后的 case。
  • agent 技术栈终于够用了:Claude Agent SDK、Meta Ads CLI、MCP Connectors。现在该出的是一个透明的参考实现,而不是又一个黑盒。

技术栈

  • 编排 — Anthropic claude-agent-sdk,supervisor 模式管理 worker 池。
  • 评测 — drip.eval:case 加载 · 打分 · LLM-as-judge · 可复现 run bundle。
  • 图像 — OpenAI gpt-image-2 出关键帧,可换。
  • 视频 — ByteDance Seedance 2.0(Volc Engine ARK),约 $0.14/秒,可换。
  • 仿真 — CAMEL-AI OASIS · 最多 100 万合成 agent,Apache-2.0。
  • 投放 — Meta + TikTok 经 MCP(pipeboard-co/meta-ads-mcp、amekala/ads-mcp)。
  • 可观测 — 自托管 Langfuse。
